# Interface: ModuleActionPagerGetterMap

从原始响应中提取分页字段时使用的字段映射，值为原始响应中的字段路径（支持 `a.b` 嵌套）。

## Properties

| Property | Type | Description |
| ------ | ------ | ------ |
| <a id="property-pageid"></a> `pageID` | `string` | 当前页码字段路径。 |
| <a id="property-recperpage"></a> `recPerPage` | `string` | 每页记录数字段路径。 |
| <a id="property-rectotal"></a> `recTotal` | `string` | 总记录数字段路径。 |
